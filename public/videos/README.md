# Video-uri Tutorial

Acest folder conține fișierele video pentru secțiunea de tutorial.

## Fișiere necesare:

### Pasul 3 - Administrare clienți
- `Adauga_client_preturi_speciale.mp4` - Video demonstrativ adăugare client și prețuri speciale
- `Adauga_client_preturi_speciale_poster.jpg` (opțional) - Imagine poster pentru video

## Cum se adaugă video:
1. Fă screen recording la funcționalitate
2. Exportă ca MP4 (format recomandat)
3. Redenumește conform listei de mai sus
4. Upload pe GitHub în acest folder
