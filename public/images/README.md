# Imagini Tutorial

Acest folder conține screenshot-urile pentru secțiunea de tutorial de pe landing page.

## Fișiere necesare:

### Pasul 1 - Setarea companiei
- `tutorial-register-1.png` - Formular date companie
- `tutorial-register-2.png` - Formular date administrator

### Pasul 2 - Produse și prețuri
- `Adauga_produs_modal.png` - Modal adăugare produs
- `Produse_adaugate.png` - Lista produse adăugate

### Pașii 3-8
Pentru celelalte pași, se pot adăuga ulterior:
- `tutorial-dashboard.png`
- `tutorial-stoc.png`
- `tutorial-comenzi.png`
- `tutorial-picking.png`
- `tutorial-foi-parcurs.png`
- `tutorial-facturare.png`

## Cum se adaugă imaginile:
1. Fă screenshot la fiecare secțiune din aplicație
2. Redenumește fișierul conform listei de mai sus
3. Upload pe GitHub în acest folder
